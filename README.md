# Informatikai Rendszer és Hálózatüzemeltető Vizsgamunka

Ez a repository az **Informatikai rendszer és hálózatüzemeltető** vizsgamunkánkhoz tartozik. A projekt során különböző hálózati eszközök konfigurációit készítettük el és dokumentáltuk.

## Tartalom

A repository jelenleg a következőket tartalmazza:

- Router konfigurációk
- Switch konfigurációk
- További hálózati eszközök konfigurációi (pl. tűzfal, hozzáférési pont)

Minden konfiguráció a megfelelő eszköz nevét és típusát jelöli a fájl nevében.
